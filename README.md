# DeepShield: Deepfake Image Detection using EfficientNetB3

## Overview

DeepShield is a deep learning-based system developed to detect manipulated and AI-generated facial images (deepfakes). The project leverages transfer learning with EfficientNetB3 and employs a two-stage training strategy to improve model robustness and generalization across multiple datasets.

The trained model is integrated into a Streamlit web application, allowing users to upload an image and receive an authenticity prediction in real time.

---

## Features

* Deepfake image detection using Convolutional Neural Networks
* Transfer learning with EfficientNetB3
* Two-stage training for improved generalization
* Real-time image classification
* User-friendly Streamlit interface
* Confidence score and inference statistics
* Lightweight deployment for demonstration purposes

---

## Methodology

### Stage 1: Initial Training

The base model was trained using transfer learning on the following dataset:

**Deepfake vs Real Images (20K Dataset)**

Dataset Link:
https://www.kaggle.com/datasets/prithivsakthiur/deepfake-vs-real-20k

The EfficientNetB3 architecture pretrained on ImageNet was used as the feature extractor. The model was trained to distinguish between authentic and manipulated facial images.

### Stage 2: Fine-Tuning for Generalization

To improve the model's ability to generalize across different deepfake generation techniques, the trained model was further fine-tuned on a second dataset:

**Deepfake and Real Images Dataset**

Dataset Link:
https://www.kaggle.com/datasets/manjilkarki/deepfake-and-real-images

This additional training phase exposed the model to a wider variety of facial manipulations and image characteristics, reducing dataset-specific bias and improving real-world performance.

---

## Model Architecture

* Base Network: EfficientNetB3
* Input Resolution: 300 × 300
* Transfer Learning: ImageNet Pretrained Weights
* Output Layer: Sigmoid Binary Classifier
* Framework: TensorFlow / Keras

### Classification Output

| Output Score | Prediction      |
| ------------ | --------------- |
| Score < 0.5  | Authentic Image |
| Score ≥ 0.5  | Deepfake Image  |

## Dataset Preparation

Images were resized to 300 × 300 pixels to match the EfficientNetB3 input dimensions.

The dataset was divided into:

* Training Set
* Validation Set
* Test Set

Data augmentation techniques were applied during training to improve model robustness and reduce overfitting.

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/DeepShield.git
cd DeepShield
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Place the trained model file in the project root directory:

```text
deepfake_modelv2.keras
```

Run the application:

```bash
streamlit run app.py
```
